# Sales-Dashboard
This dashboard offers a streamlined interface for visualising data, tracking key metrics, and managing user interactions. It features a sidebar navigation, interactive cards, charts, tables, and theme toggles. Perfect as a foundation for an admin panel, analytics portal or internal business management tool.
# Dataset used


## Key Questions / KPIs

- Compare Sales vs Orders in a single visual chart.
- Identify the month with the highest Sales and Orders.
- Analyze Sales distribution by Gender.
- Explore Order Status distribution for 2022.
- List Top 10 States contributing most to Sales.
- Study the relation between Age Group and Gender based on order count.
- Determine which Sales Channel contributes maximum revenue.
- Find the highest-selling Product Category.
- Calculate Percentage of Total Orders Delivered.


##Process

Data Cleaning:
Verified for missing, inconsistent, or duplicate entries.
Standardized date formats, column headers, and numeric types.
Removed anomalies and ensured uniform units.

Data Processing:
Created Pivot Tables to answer each KPI.
Used calculated fields to derive additional insights such as profit %, average order value, etc.
Designed conditional formatting for better visual readability.

Dashboard Development:
Combined all Pivot Tables and Charts into a single interactive dashboard.
Added Slicers and Timelines to filter data dynamically (by Year, State, Gender, Channel, and Category).
Enhanced layout using Excel Shapes, Icons, and consistent color themes for a professional look.
